3.4. 圖示字型 (Glyphicons)

將圖示放到網頁中是很容易的，如以下範例：

<i class="i-colorgy"></i>

或是應用稱作 ligature 的印刷排版技術，您也可以這樣使用圖示：

<i class="icon">colorgy</i>

colorgy

透過 ligature 技術可以讓圖示的文字名稱直接顯示為圖示本身，提供了更好的瀏覽體驗 (例如可被全文搜尋、優雅降級成顯示文字、可被文字閱讀器讀用等)。

或是直接使用 UTF-8 code 也可以：

<p class="i-icon">&#x19a28;</p>
